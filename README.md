# Assembly Documentation Formater
Makes nice looking formated assembly documentation. I use it for MIPS documentation.

## Example:
<pre>
############################################
#             Binary Search               #
############################################
# Input:                                   #
# $a0, array                               #
# $a1, left index                          #
# $a2, right index                         #
# $a3, item to search for                  #
############################################
# Output:                                  #
# $v0, index of the item (-1 if it failed) #
############################################
# Used Registers:                          #
# $t0, mid point                           #
# $t1, used in math                        #
############################################
</pre>
